Description of the Assignment:

Today, I was assigned to build a Logic Builder program using Python.
The task was to print numbers from 1 to 50 using Fizz/Buzz logic:

Print “Fizz” if the number is divisible by 3
Print “Buzz” if divisible by 5
Print “FizzBuzz” if divisible by both 3 and 5
Otherwise, print the number

Additionally, I had to count how many times Fizz, Buzz, and FizzBuzz occurred using loops and functions.

This assignment helped strengthen my logical thinking and understanding of loops, conditions, and functions.
