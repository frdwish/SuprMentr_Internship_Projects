Assignment Name: Student Score Predictor

Internship Diary Entry
Description of the Assignment:

On this day, I worked on building a Student Score Predictor using basic Machine Learning concepts.

The objective of this assignment was to understand how prediction models work by using simple student data. I created a small dataset containing study hours and corresponding test scores, then applied a Linear Regression model to predict student performance.

The program was able to:

Train a model using existing data
Predict scores based on study hours
Test predictions using new input values

This assignment helped me understand the fundamentals of prediction systems and how Machine Learning models identify patterns in data.
