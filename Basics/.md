Internship Diary – First Project

Description of the Assignment:

On this day, we were given a basic programming assignment to improve our understanding of Python fundamentals.
The task included three parts:

Writing a program to check whether a given number is odd or even.
Printing numbers from 10 to 1 using a loop.
Creating a program that keeps asking the user to enter numbers and calculates the sum, stopping only when the user enters 0.

This assignment helped me practice conditional statements, loops, and user input handling in Python.
