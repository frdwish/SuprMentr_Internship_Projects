Assignment Name: KNN in Real Life

Internship Diary Entry
Description of the Assignment:

On this day, I worked on understanding how the K-Nearest Neighbors (KNN) algorithm is used in real-life recommendation systems like Netflix.

The objective of this assignment was to learn how similarity between users or items helps in generating personalized recommendations.

I studied how platforms compare user preferences such as watched movies, ratings, and interests. Using KNN, the system finds users with similar tastes and recommends movies that those similar users liked.

To understand the concept better, I created a small example that measures similarity between data points and identifies the nearest neighbors.

This assignment helped me understand how distance-based algorithms work and how ML is applied in entertainment platforms.
