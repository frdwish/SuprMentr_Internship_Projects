Description of the Assignment:

Today, I was assigned to create a Student Data Manager program using Python.
The task was to store data for 5 students using dictionaries. The program should:

Store student names and marks
Calculate the class average
Find the topper (student with highest marks)
Assign grades based on marks

This assignment helped me understand how dictionaries work and how to manage structured data efficiently.


Learning Outcomes:
Improved understanding of dictionaries in Python.
Learned how to retrieve maximum values using built-in functions.
Practiced calculating averages using loops and aggregation functions.
Strengthened logical thinking in grading systems.
