Assignment Name: Build Your First Dataset

Internship Diary Entry
Description of the Assignment:

On this day, I worked on creating my first simple dataset using Python.
The objective of this assignment was to understand how datasets are structured and how they are used in Machine Learning.

For this task, I created a dataset representing the relationship between study hours and marks obtained by students. After creating the dataset, I identified the features (input variables) and labels (output variable).

Feature: Study Hours
Label: Marks

By observing the dataset, I tried to understand the relationship between the two variables. The general trend shows that as study hours increase, the marks also tend to increase, which represents a simple predictive relationship often used in basic machine learning models.

This assignment helped me understand the basic concept of datasets, features, labels, and how data can be used to make predictions.


Learning Outcomes:
Learned how datasets are created and structured.
Understood the difference between features and labels.
Learned how data relationships help in prediction tasks.
Gained basic understanding of data preparation for machine learning.
