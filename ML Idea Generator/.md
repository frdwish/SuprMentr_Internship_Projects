Description of the Assignment:

In this assignment, I was asked to generate Machine Learning problem ideas in different domains such as:

College
Healthcare
Shopping

I had to clearly describe the input → output for each idea.

This task helped me think practically about how ML models are applied in real-world situations.

ML Ideas with Input → Output:
1. College Attendance Predictor

Input → Student attendance records, previous performance
Output → Risk prediction (Pass/Fail probability)

2. Student Placement Predictor

Input → CGPA, skills, internship experience
Output → Probability of getting placed

3. Disease Prediction System

Input → Symptoms, medical history, test results
Output → Predicted disease type

4. Medicine Recommendation System

Input → Patient diagnosis, age, allergies
Output → Suggested medicines

5. Product Recommendation System

Input → User browsing history, purchase history
Output → Personalized product suggestions

6. Price Prediction for Shopping

Input → Past product prices, demand, season
Output → Predicted future price
